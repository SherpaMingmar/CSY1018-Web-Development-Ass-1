Before HTML5, a video could only be played in a browser with a plug-in (like flash). The HTML5 video element specifies a standard way to embed a video in a web page. The controls attribute adds video controls, like play, pause, and volume. It is a good idea to always include width and height attributes. If height and width are not set, the page might flicker while the video loads.
